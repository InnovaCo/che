#### *module* clicks/forms #

Обрабатывает клики на кнопке submit для форм с аттрибутом data-reload-sections, достает данные о необходимых секциях, url и вызывает обработчика, после чего отменяет дефолтное поведение. Обрабатывает именно клики по кнопке, так как необходима делегация событий, а событие submit не поднимается вверх к корню.

Непосредственно навешивание обработчика, который работает только если есть хоть один обработчик клика, на это указывает наличие clicks

dom('body').on "form[#{config.reloadSectionsDataAttributeName}] input[type='submit']", "click", (e) -> if clicks?

Достаем форму из родителей кнопки

Достаем метод по которому должна быть отправлена форма, по умолчанию это "GET". Следует отметить, что "POST"-запросы за секциями не кешируются
