¶

Непосредственно навешивание обработчика, который работает только если есть хоть один обработчик клика, на это указывает наличие clicks

 
  dom('body').on "a[#{config.reloadSectionsDataAttributeName}]", "click", (e) ->
    if clicks?
      
      data = @getAttribute config.reloadSectionsDataAttributeName
      url = @getAttribute 'href'
      clicks.trigger "anchor:click", [url, data, "GET"]

      e.preventDefault()
      return false

init(callback)

Интерфейс модуля. Представляет функцию, которая принимает обработчиков кликов. Функция создает новый диспетчер событий, если до этого его не было

 
  init = (callback) ->
    if not clicks?
      clicks = events.sprout "anchors"

    clicks.bind "anchor:click", callback
