    module clicks/forms

    Обрабатывает клики на кнопке submit для форм с аттрибутом data-reload-sections, достает данные о необходимых секциях, url и вызывает обработчика, после чего отменяет дефолтное поведение. Обрабатывает именно клики по кнопке, так как необходима делегация событий, а событие submit не поднимается вверх к корню.

    init(callback)

    Интерфейс модуля. Представляет функцию, которая принимает обработчиков кликов. Функция создает новый диспетчер событий, если до этого его не было

      init = (callback) ->
    if not clicks?
      clicks = events.sprout()

    clicks.bind "form:click", callback
